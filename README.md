# about

blurbattle is a simple plugin for pvp duels. it uses the multiverse-core plugin to have a seperate world for duels. this plugin is perfect for smp's and other small servers.

## building

Like any java plugin, just run
```shell
./gradlew build
```
or
```batch
.\gradlew.bat build
```
